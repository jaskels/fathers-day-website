# Happy Father's Day Website

This is a simple static website.

## How to open it
1. Open the folder.
2. Double-click `index.html`.

## Files
- `index.html` = the website page
- `styles.css` = the design
- `assets/father-fishing-lake.png` = the background image

## How to put it online
You can upload these files to Netlify, Bluehost, GoDaddy, or any basic web hosting account.
